🤖 Foundry Agent Bot

Bot Framework wrapper around Microsoft Foundry agents — chat in Teams or Web Chat.

Endpoints

  • /health — liveness check
  • /admin/chat — 🧪 test chat with an agent in your browser
  • /admin/manifest — generate & download Teams app manifest
  • /admin/agents — list registered Foundry agents (JSON)
  • /admin/registrations — bot ↔ agent registrations (JSON)
  • POST /api/messages — Bot Framework messaging endpoint
  • POST /api/notify — proactive message push

How to use

  1. Open /admin/manifest and download an agent's zip.
  2. Sideload it in Teams (Apps → Manage your apps → Upload a custom app).
  3. Open a 1:1 chat with the bot. Try /help to see commands.
